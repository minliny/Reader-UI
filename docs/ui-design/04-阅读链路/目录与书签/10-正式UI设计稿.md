## 定义范围

本页面包定义阅读控制层里的目录/书签快捷模块。当前 P0 以本地 demo 的 `toc-bookmarks` 模块为准；完整目录页属于书籍详情链路的独立页面，不从快捷模块里强制跳转。

## 结构

- `QuickModule`：替换阅读控制层左侧主内容区，只显示目录/书签切换、紧凑章节列表和右侧下载/书签状态图标。
- `DirectoryRows`：章节行以章节名为主，右侧保留下载状态和书签状态两个图标槽位；不显示摘要、更新时间、更多菜单或完整目录入口。
- `BookmarkRows`：书签标签复用同一容器和行高规则，保持底部模块导航、右侧亮度栏和阅读正文位置不变。

## 继承规则

- 必须继承阅读控制层顶部条、底部面板、右侧亮度栏和底部模块导航。
- QuickModule 不扩大底部面板。
- 快捷模块不展示“完整目录”按钮；完整目录页只能从书籍详情目录入口进入。

- 不把目录层级重复写入每一行章节标题。
