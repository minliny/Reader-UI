## 定义范围

本页面包定义阅读控制层里的设置快捷模块。当前 P0 以本地 demo 的 `reader-settings` 模块为准，设置内容面向阅读行为，不负责书源、RSS、同步或账号。

## 结构

- `QuickModule`：自动翻页、点击翻页方式、音量键翻页、翻页动画、横屏锁定、屏幕常亮、页脚进度信息、触摸反馈、自动缓存后续章节。
- `ValueRows`：点击翻页方式和翻页动画使用右侧当前值与箭头；开关项使用右侧 switch。
- `ModuleNav`：目录、朗读、界面、设置四个入口尺寸和位置保持稳定，设置按钮仅改变选中态。

## 继承规则

- QuickModule 只替换阅读控制层左侧主内容区。
- 快捷模块不展示“更多阅读设置”或“完整设置”入口。
- 设置项使用当前阅读控制层列表样式，右侧亮度栏和底部模块导航不移动。
