## 定义范围

本页面包定义阅读控制层里的界面快捷模块。当前 P0 以本地 demo 的 `reader-appearance` 模块为准，完整字体、主题、版式和翻页动画管理页只作为后续扩展。

## 结构

- `QuickModule`：阅读主题色块、字号、行距和字体等核心控件。
- `ThemeSwatches`：色块只表现主题颜色与选中态，不在色块内加入额外图标。
- `TypographyRows`：字号、行距等参数使用当前值和加减按钮，控件比例以 `UI设计图.png` 为准。

## 继承规则

- QuickModule 只替换阅读控制层左侧主内容区。
- 右侧亮度栏常驻。
- 快捷模块不展示完整管理页入口；右侧亮度栏和底部模块导航不移动。
